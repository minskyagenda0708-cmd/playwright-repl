Dramaturg

Playwright engineer's companion — console, editor, runner, and recorder in a Chrome side panel

Getting Started

  1. Click the Dramaturg icon to open the side panel
  2. Navigate to a webpage — it auto-attaches
  3. Use keyword commands or Playwright API in the console
  4. Record actions, pick elements, or write scripts in the editor

Quick Reference

gotoNavigate to URL

snapshotShow accessibility tree
screenshotCapture page image
selectChoose dropdown option

helpShow all commands
